## 📋 TEMPLATE PROMPT CHO AI STUDIO

### Prompt Cơ Bản (Copy và sử dụng):

```
Phân tích hình ảnh UI này và tạo React component với yêu cầu:

🎯 KỸ THUẬT:
- React 18+ functional component với hooks
- TypeScript (nếu có thể)
- Tailwind CSS v3 cho styling
- Lucide React cho icons
- Responsive: mobile-first (320px, 768px, 1024px+)

🎨 THIẾT KẾ:
- Màu sắc chính xác theo hình (hex codes)
- Font: Inter hoặc tương tự
- Spacing: hệ thống 4px/8px
- Shadows, borders, radius theo hình
- Hover effects và transitions

📦 CẤU TRÚC:
- Import statements đầy đủ
- Props interface/types
- Component chính
- Sub-components nếu cần
- Export statement

✨ TÍNH NĂNG:
- Accessibility (ARIA, semantic HTML)
- Loading states
- Error handling
- Comments giải thích bằng tiếng Việt

Xuất ra code production-ready, có thể chạy ngay.
```

### Git Workflow:
```bash

git checkout -b feature/集运服务-quang

# Commit thường xuyên
git add .
git commit -m "feat: thêm component 集运广场"

# Tạo Pull Request khi hoàn thành
```

### Naming Convention:
- **Components:** PascalCase (VD: `JiYunGuangChang.jsx`)
- **Files:** kebab-case (VD: `ji-yun-guang-chang.jsx`)
- **Variables:** camelCase (VD: `packageList`)
- **Constants:** UPPER_SNAKE_CASE (VD: `MAX_ITEMS`)
