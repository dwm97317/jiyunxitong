# 集运系统 UI 截图分类

本目录包含集运系统的所有 UI 截图，按功能模块分类为 4 个文件夹：

## 📁 文件夹结构

### 01-首页查询模块 (15 张)
- 首页相关：个人件、集运件、整柜直达
- 查询功能：默认状态、输入中、查询结果、查询详情
- 快递详情、扫码取件等

### 02-寄件物流模块 (36 张)
- 寄件流程：填写收件人、选择物流方式
- 物流选项：快递、物流、一票一件、一票多件
- 特殊服务：特快件、标块件、大票直达
- 费用相关：价格时效、运费测算、计费规则
- 网点服务：线下网点、网点自提
- 规则说明：收寄标准、违禁物品、服务协议

### 03-集运服务模块 (17 张)
- 集运功能：集运广场、集运详情
- 包裹管理：包裹预报、包裹认领、申请打包
- 增值服务：保险、在线理赔、理赔流程
- 付款相关

### 04-用户中心模块 (28 张)
- 个人中心：我的、我的钱包、我的快递
- 分销系统：我的分销、团长中心、我的团队
- 拼团功能：发起拼团、加入拼团
- 邀请系统：邀请记录、佣金明细、返佣订单
- 账户管理：地址簿、实名认证、绑定邮箱
- 其他功能：客服、站内信、功能反馈、福利、好友寄、代付

## 📊 统计信息
- 总计：96 张 PNG 截图
- 分类：4 个功能模块
- 覆盖：完整的集运系统业务流程
