# 集运中心 App - 设计系统

## 1. 品牌定位

物流集运服务移动应用，专业、高效、可信赖

## 2. 视觉风格

- 现代简约
- 移动优先设计
- 清晰的信息层级
- 友好的交互反馈

## 3. 色彩系统

### 主色调
- Primary: `#1677FF` (蓝色 - 主要操作)
- Success: `#52C41A` (绿色 - 成功状态)
- Warning: `#FAAD14` (橙色 - 警告提示)
- Error: `#FF4D4F` (红色 - 错误状态)

### 中性色
- Text Primary: `#000000` `#1F1F1F`
- Text Secondary: `#666666` `#999999`
- Border: `#E5E5E5` `#F0F0F0`
- Background: `#FFFFFF` `#F7F8FA`

## 4. 字体系统

- 标题：16-20px, font-weight: 600
- 正文：14px, font-weight: 400
- 辅助文字：12px, font-weight: 400
- 字体族：-apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif

## 5. 组件规范

### 按钮
- 主按钮：蓝色背景，白色文字，圆角 8px
- 次要按钮：白色背景，蓝色边框，圆角 8px
- 高度：44px（移动端触摸友好）

### 卡片
- 背景：白色
- 圆角：12px
- 阴影：subtle shadow
- 内边距：16px

### 输入框
- 高度：44px
- 圆角：8px
- 边框：1px solid #E5E5E5
- 聚焦：蓝色边框

### 导航栏
- 高度：44px
- 背景：白色
- 底部边框：1px solid #F0F0F0

### 标签页
- 高度：50px
- 激活状态：蓝色文字 + 底部指示条

## 6. Stitch 提示设计系统块

**设计系统（必需）：**

使用移动优先的现代简约风格。色彩：主色 #1677FF（蓝色），成功 #52C41A，警告 #FAAD14，错误 #FF4D4F。中性色：文字 #000/#666/#999，边框 #E5E5E5，背景 #FFF/#F7F8FA。

组件规范：
- 按钮高度 44px，圆角 8px，主按钮蓝底白字
- 卡片圆角 12px，白色背景，subtle shadow，内边距 16px
- 输入框高度 44px，圆角 8px，边框 #E5E5E5
- 导航栏高度 44px，白色背景
- 字体 14px 正文，16-20px 标题，12px 辅助文字

布局：移动端宽度 375px，内容区域左右边距 16px，组件间距 12-16px。使用 Tailwind CSS 实现响应式设计。

**重要：所有界面文字必须使用简体中文，包括按钮、标签、提示文字、导航等所有文本内容。**
